Bill PDF Generator

API Endpoints for PDF Generation:

GET /api/bills/{id}/pdf - Generate PDF by Bill ID

GET /api/bills/number/{billNumber}/pdf - Generate PDF by Bill Number

Example Usage:

http://localhost:8081/bookshop/api/bills/1/pdf

http://localhost:8081/bookshop/api/bills/number/BILL202401001/pdf
